# prettier-config

1. Install the package:
    - Using npm:
      ```shell
      npm i -D @incloud/prettier-config

    - Using yarn:
      ```shell
      yarn add -D @incloud/prettier-config
      ```

2. Create `.prettierrc` with the following content:

    ```json
    "@incloud/prettier-config"
    ```
